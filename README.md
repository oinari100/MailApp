## 概要
メールの例文を検索するのが面倒だったため、いい感じにテンプレートから作成してくれるアプリを作成

※ リリースなし

#### 技術スタック
- React Native（expo）
- Firebase

#### 内容
- 検索欄や一覧から自身が作成したいメール内容（遅刻や早退など）を選ぶ
- 登録しておいた自身の登録内容（名前、所属会社など）に沿った、メール件名、メール本文でメーラーが立ち上がる
- そのまま送信して終了
